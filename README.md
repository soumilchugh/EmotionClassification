# Emotion Classification

This repository contains a computer-vision project for facial emotion classification experiments.

The project is kept public as part of a broader machine-learning and computer-vision portfolio. It is useful as a compact example of image-classification work and model-evaluation workflow.

## Focus Areas

- Image classification
- Facial-expression recognition
- Computer-vision model training
- Evaluation of predicted emotion classes

## Portfolio Context

Emotion Classification sits alongside related visual-perception projects such as lane detection, pupil center estimation, retinal image analysis, and medical-image classification.

## Notes

This is an older research and experimentation repository. The README is intentionally concise until setup instructions, dataset notes, and example outputs are added.
